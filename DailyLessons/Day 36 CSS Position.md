# Day 36 - CSS Position

# CSS position property

We can display items using certain different mechanisms

    position : fixed
    
    position : absolute
    
    position : relative

    div.fixed{
        position:fixed;
        top:0;
        left:0;
    }
    /* Absolute position is relative to the container directly */
    div.absolute{
        position:absolute;
        margin-top:100px;
    }
    div.special{
        margin-top:100px;
        background-color:#f44;
    }
